# nic-crds

## Description

Packages the full set of CRDs for NGINX Ingress Controller from upstream NGINX repository, so that they can be managed
independently from the NGINX OSS/NGINX+ Ingress Controller.

## Usage

### Fetch the package

```shell
kpt pkg get --for-deployment https://github.com/memes/google-demo-helper-modules.git/applications/nic-crds nic-crds
```
